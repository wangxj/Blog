Enum 类型介绍

枚举类型(Enumerated Type)
很早就出现在编程语言中，它被用来将一组类似的值包含到一种类型当中。而这种枚举类型的名称，则会被定义成独一无二的类型描述符。
这一点的定义和常量类似，不过相比较常量类型，枚举类型可以为申明的变量提供更大取值范围。

定义一个枚举

为了改进 Java 语言在这方面的不足弥补缺陷，5.0 版本 SDK 发布时候，在语言层面上增加了枚举类型。枚举类型的定义也非常的简单，用 enum 关键字加上名称和大括号包含起来的枚举值体，像类那样定义即可。

另外，java平台也为Enum类型提供了高质量的实现，比如默认实现 Comparable 和 Serializable 接口，让开发者一般情况下不用关心这些细节。
